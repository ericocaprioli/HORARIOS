# Upload de PDFs

Coloque seus arquivos PDF nesta pasta com os nomes exatos usados em `pdfs.js`.

- Exemplo: `1A.pdf`, `2A.pdf`, `atividades-adrieli.pdf`
- Se você enviar um novo arquivo com o mesmo nome, ele substituirá o PDF antigo.
- Não altere o nome em `pdfs.js` sem atualizar também o arquivo correspondente nesta pasta.
